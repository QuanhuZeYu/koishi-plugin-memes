## 安装指南 🛠️

浏览器搜索安装`FFMPEG`并设置好环境变量即可享用

## 用法

使用 `petpet` 不带参数用法就是生成一张摸自己头的meme

使用 `petpet` + `@任意群成员` 生成一张摸该成员头像的meme

使用`petpet` + `图片` 生成摸该图片的meme
